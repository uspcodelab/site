  1. Aula 1

    Para 18/04

    Faça a sua versão da página feita pelo Rodrigo, com seus filmes favoritos!

    • Adicione filmes seguindo os seguintes critérios:
      • uma lista com vários filmes usando a tag <ul> (unordered list), criando novos itens com a tag <li> (list item);
      • coloque o título dentro de uma tag de heading <h3>;
      • além disso, o título do filme deve ser um hyperlink que direciona para a página correspondente em imdb.com, para isto basta usar a tag <a>(anchor) colocando a referência da página no atributo href;
      • inclua uma breve sinopse para cada filme usando a tag <p> (paragraph);
      • padronizar o tamanho de cada imagem usando o atributo width (assim como o src) da tag <img> (image).
    • Sempre que tiverem dúvidas perguntem no nosso grupo do telegram!

  2. Aula 2

    Para 25/04

    Nessa tarefa vamos começar a utilizar HTML semântico e usar algumas regras de CSS para que vocês comecem a se familiarizar com elas! Para começar vamos reutilizar a tarefa da aula passada como base.

    • Primeiramente, além de listar seus filmes favoritos, crie pelo menos mais duas listagens de favoritos como livros, hobbies, matérias etc. Feito isto, vamos criar um mecanismo de navegação interna para cada item de favorito.
      1. Crie um bloco com a tag semântica <section> para cada seção de favoritos, acrescente o atributo id="nome_do_bloco" de acordo com o nome da seção de favoritos, id="filmes" por exemplo.
      2. Para cada título de favorito, Filmes por exemplo, adicione-o dentro de uma tag <p>, inserindo o atributo class="titulo_favorito", esse atributo se mantem o mesmo para todos os nomes de seção de favoritos (essa classe usaremos para personalizar com CSS mais tarde);
      3. Feito isso, as seções estão cada um com seu id próprio, assim podemos criar a navegação. Para isto, crie uma tag <nav> no início da página que englobe os itens com os nomes dos títulos dentro de uma tag <a>. Em vez de referenciar uma página externa dentro do atributo href, insira uma referência para a tag usando um seletor de id, por exemplo, <a href="#filmes">Filmes</a>.
      4. Pronto, temos uma navegação interna no início da página!
    • Começando um pouco com estilização, vamos aprender como trocar a fonte de toda a nossa página, para isto, entre no Google Fonts para escolher uma fonte e siga os seguintes passos:

      1. Procure uma fonte desejada (no caso eu escolhi a Roboto Mono);

      2. Selecione um estilo da fonte;

      3. Clique no botão localizado no canto superior direito para exibir a fonte que você selecionou;

      4. Clique em Embed e copie tudo que está dentro da tag <link> e cole dentro do <head> da sua página. Além disso, dentro de <style> crie uma nova regra de CSS para a tag <html> (assim toda página irá usar a nova fonte) e cole a regra font-family lá.

      5. Obs.: Existem outras alternativas para adicionar fonte ao seu site, acima é citado apenas uma das possibilidades.
    • Agora que temos uma página com bastante conteúdo e com uma fonte maneira, podemos começar a inserir estilos aos elementos da página em si. Usando o que foi ensinado em aula, sinta-se livre para usar sua imaginação. Abaixo listamos alguns aspectos e dicas de quais regras você pode modificar!
      • Textos: podemos modificar os títulos dos nossos favoritos que anteriormente foram marcados com a classe titulo_favorito, modificando aspectos como tamanho de fonte (jeito certo de modificar tamanho de fonte em vez de usar tags h), peso da fonte (jeito certo de fazer negrito sem tag strong ) espaçamento entre as letras, alinhamento de texto, cor do texto etc. 
          .titulo_favorito {
    font-size: 3em;
    font-weight: 800;
  }
      • Box Model: podemos modificar enchimento, borda e margem dos elementos da nossa página. Assim, podemos adicionar por exemplo, espaçamento entre cada item de lista li usando 
          li {
    margin: 1em 0;
  }
        Uma questão importante deste ponto, se da em relação à importância de cada detalhe e nos seus impactos no visual da página. Podemos pensar no espaçamento adequado entre elementos, cor da borda, no raio da borda, no sombreamento do elemento etc. Por isso, vale a pena entender bem os conceitos daqui e saber usa-los corretamente.
      • Cores e Tamanho dos elementos: pode ser bacana brincar um pouco com a questão de tamanho e cor dos elementos, para isso você pode modificar as seguintes regras:
        • Largura
        • Altura
        • Cor de fundo
      • Pseudoclasses: existem várias, elas atuam de forma a especificar um comportamento especial para um dado elemento. Na barra de navegação que fizemos anteriormente, poderiamos adicionar a pseudoclasse hover para agir sobre cada tag <a>, assim quando você passar o mouse por cima de cada referência ele poderia mudar de cor por exemplo. Para isto, adicione uma classe dentro de cada tag <a class="nav_item" href="#filmes">, agora dentro da sua folha de estilos faça: 
          nav_item:hover {
    background-color: red;
  }
      • Pretendemos continuar com mais algumas aulas sobre CSS mas vale lembrar que isso é só a ponta do iceberg. Existem muitas outras regras para cada aspecto citado e é muito importante que vocês comecem a se familiarizar à busca-los no MDN ou outra fonte à sua escolha. Outro ponto, é que pouco importa decorar todos as regras (existem muitas), o mais importante é compreender como usar corretamente cada uma que você aprende, e ir descobrindo novas regras com o tempo de acordo com as suas necessidades!
    • Sempre que tiverem dúvidas perguntem no nosso grupo do telegram!
